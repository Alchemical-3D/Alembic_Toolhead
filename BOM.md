# Bill of Materials (BOM)

This list contains all the 3D-printed and physical hardware required to assemble the Alembic Toolhead.

*Note: Quantities for some hardware fasteners (e.g. M3 screws, heat inserts) are estimates and may be slightly higher/lower based on your exact A3DP FXD extruder/carriage configuration.*

## 1. Printed Parts
Print these out using the guidelines specified in the [Print Settings](Print_Settings.md) page.

| Part File Name | Qty | Description / Notes |
| --- | :---: | --- |
| `Lower_Body.stl` | 1 | Main lower chassis. |
| `Manifold_Frame.stl` | 1 | Pneumatic cooling mechanisms. |
| `Fan_Baffle_Takoto.stl` | 1 | Specifically for the Takoto. |
| `Chainflex_Clamp_Main.stl` | 1 | Top clamp for the electrical umbilical. |
| `Chainflex_Clamp_Base.stl` | 1 | Base support for the electrical umbilical. |
| `Pnuematic_Strain_Relief.stl`| 1 | Secures the pneumatic tube. |
| `Sensor_Retainer.stl` | 1 | Retains BDensor. (optional) |
| `BD_Mod_Clamp.stl` | 1 | Mod clamp for BDsensor. (optional) |
| `Shim_06mm_BDsensor.stl` | 1 | 0.6mm shim for BDsensor offset. (optional) |
| `Bend_Tool_Base.stl` | 1 | Jig tool base for bending tubing. |
| `Bend_Tool_Lever.stl` | 1 | Jig tool lever for bending tubing. |
